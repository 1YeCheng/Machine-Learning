# Lab 2: SVM and SVR Experiments

This lab contains SVM/SVR experiments, LIBSVM-based workflows, and multi-model classification comparisons.

## Contents

- `data/`: Watermelon, Wine, and WDBC datasets.
- `src/`: Experiment pipeline and a handwritten C4.5 decision tree implementation.
- `results/`: Metrics, predictions, model files, and visual outputs.
- `libsvm-3.31/`: Bundled LIBSVM source and Windows executables.

## Run

```bash
python src/run_lab2_experiments.py
```
